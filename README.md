Jarvis Voice Assistant (Python)

A simple voice-controlled assistant built using Python that can perform basic tasks like opening websites, playing music, performing calculations, and more.

🚀 Features
🎤 Voice Recognition using speech_recognition
🔊 Text-to-Speech using pyttsx3
🌐 Open websites (Google, Facebook, Instagram, WhatsApp)
🎵 Play songs from custom music library
➕ Perform basic addition
🎮 Simple number guessing game
🤖 Voice-triggered command system ("Monu")
🧠 Continuous listening loop
🛠️ Technologies Used
Python
SpeechRecognition
Pyttsx3
OpenCV (cv2)
Requests
Webbrowser
Random Module
